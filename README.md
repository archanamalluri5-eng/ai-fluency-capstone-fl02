# AI Fluency Capstone

This repository contains my capstone project for the FlyRank AI Fluency Internship.

## Tech Stack
- React
- JavaScript
- HTML
- CSS
## Features
- AI-assisted development
  
## What This Project Does

This project is part of the FlyRank AI Fluency Internship capstone. It demonstrates AI-assisted development practices, Git version control using Conventional Commits, and project documentation. The repository showcases the setup of an AI development environment and responsible collaboration with AI tools.
